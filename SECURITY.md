# Security Policy — AlephOne

## Versiones soportadas
AlephOne está en desarrollo activo. Solo la versión más reciente recibe actualizaciones de seguridad.

## Reportar una vulnerabilidad
Si encontrás una vulnerabilidad, mandá un mail a [lucas6csta@gmail.com] con:
- Descripción del problema
- Pasos para reproducirlo
- Impacto potencial

Respondemos en menos de 7 días. No publiques vulnerabilidades públicamente antes de que las hayamos corregido.
